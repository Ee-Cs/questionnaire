Feedback Questionnaires Manager

The RESTful implementation with JpaRepository and H2 database

1. The endpoints generated by Spring

Get all Questionnaires /questionnaires
Get the Questionnaire by id /questionnaires/1
Get the Inbox in Questionnaire by id /questionnaires/1/inboxes/11
Get all Respondent Replies in Inbox by id /inboxes/11/respondentReplies
Get all Inboxes /inboxes
Get the Inbox by id /inboxes/11
Get all Respondent Replies /respondentReplies
Get the Respondent Reply by id /respondentReplies/111
Endpoints from the InboxRepository interface /inboxes/search
Get the Inboxes by topic with word 'team' /inboxes/search/findByTopicContaining?word=team
Count the Inboxes by topic with word 'team' /inboxes/search/countByTopicContaining?word=team
Count the Inboxes with anonymous submission allowed /inboxes/search/countByAnonymousSubmissionAllowed?anonymousSubmissionAllowed=true
Get the expired Inboxes /inboxes/search/findAllWithExpirationDateBeforeNow
Endpoints from RespondentReplyRepository interface /respondentReplies/search
Get the Respondent Replies from respondent 'bob' /respondentReplies/search/findByRespondentSignatureContaining?username=bob
Get the Respondent Replies from respondent 'charlie' /respondentReplies/search/findByRespondentSignatureContaining?username=charlie

2. The endpoints from controllers

Read Questionnaire as questioner /read_questionnaire_as_questioner_path?id=1&ownerSignature=UNKNOWN
Read Questionnaire as respondent /read_questionnaire_as_respondent_path?id=1
Create report for respondent 'bob' /report?ownerName=alice&respondentName=bob
Create report for respondent 'charlie' /report?ownerName=alice&respondentName=charlie
Create report for anonymous respondent /report?ownerName=alice&respondentName=
